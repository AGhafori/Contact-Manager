# Contact-Manager

This is an Android Application that will save and edit a contact in the application.
The user will be able to add Name, Last Name, phone number, date of birth, date they meet and address of the contact.
These data will then be saved once the save button is hit and can be brought up when the contact is clicked in the listview.

The user can map the address of the contact and determine the distance between the user and the address given and can also get the lng and lat.
